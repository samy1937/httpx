## 简介

> 判断http/https并截图

## Usage

```bash
└> ./httpx
Usage of ./httpx:
  -chrome string
    	chrome path
  -output string
    	output file name (default "202105292011.html")
  -proxy string
    	probe proxy, example: http://127.0.0.1:8080
  -target string
    	target ip:port
  -targets string
    	target ip:port list file
  -thread int
    	probe thread (default 10)
  -timeout int
    	probe http request timeout (default 10)
```
